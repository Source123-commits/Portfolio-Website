# Sakib Ahmed | Software Developer Portfolio

A modern, responsive personal portfolio website showcasing projects, skills, and professional information.

## Features

- **Responsive Design** - Mobile-friendly layout that adapts to all screen sizes
- **Modern Navbar** - Fixed navigation with smooth scrolling links
- **Hero Header** - Eye-catching introduction section with call-to-action buttons
- **About Section** - Professional bio with statistics
- **Skills Section** - Display of technical skills and competencies
- **Projects Portfolio** - Showcasing completed projects with descriptions
- **Contact Form** - Get in touch section for inquiries
- **Smooth Scrolling** - Enhanced user experience with smooth navigation

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No build tools or dependencies required - it's pure HTML, CSS, and JavaScript

## Customization

### Update Personal Information
Edit `index.html` to change:
- Your name and title
- About me description
- Projects and their details
- Contact information

### Modify Colors
Edit `styles.css` CSS variables:
```css
:root {
  --bg: #0b1220;           /* Background color */
  --accent: #7c3aed;       /* Accent color */
  /* ... other variables */
}
```

### Add New Projects
Add new project cards in the projects section of `index.html`
